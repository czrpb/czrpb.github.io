### Foundations: Functions as Vaules

A function:

- Is a thing/type/object just like any other: `int`, `str`, `class <OBJ>`
- Is referenced by a name: `def <NAME>` or `<NAME> = lambda ...`

Thus, passed like any other type:

```python
def add1(x):
    return x+1
    
def adfunc(x):
    return add1(add1(x))

def apply(func, x):
    return func(x)
    
>>> apply(add1, 7)
8
>>> apply(adfunc, 7)
9
>>> apply(add1, apply(add1, 7))
9
```

### Decorator: In words?

A decorator "surrounds" a function:

```
    P-ARGS     V-ARGS     KW-ARGS
      |          |           |
+=====|==========|===========|=====+
|     v          v           v     |
|                                  |
|             C O D E              |
|                                  |
|          +--v--v--v--+           |
|          |  C O D E  |           |
|          +-----v-----+           |
|                                  |
|             C O D E              |
|                                  |
|================|=================|
                 |
                 v
               OUTPUT
```

---

### Decorator

```python
def <NAME>(<P-ARGS>, *<V-ARGS>, **<KW-ARGS>):

    def <NAME>(<P-ARGS>, *<V-ARGS>, **<KW-ARGS>):

        <CODE>
        
        return <VALUE>

    return <VALUE>
```

### Decorator: `double`

```python
def double(x):

    print("Doubling ...")
    
    result = x*2
    
    print(f"\tDone doubling: {result}")
    
    return result
```

---

### Decorator: `double` cont

```python
def _double(x):
    return x*2
    
def double(x):

    print("Doubling ...")
    
    result = _double(x)
    
    print(f"\tDone doubling: {result}")
    
    return result
```

---

### Decorator: `double` cont

```python
def _double(x):
    return x*2
    
def double(x, func=_double):

    print("Doubling ...")
    
    result = func(x)
    
    print(f"\tDone doubling: {result}")
    
    return result
```

---

### Decorator: `double` cont

```python
def _double(x):
    return x*2
    
def double(x, func=_double):

    def wrapper(x):
        print("Doubling ...")
    
        result = func(x)
    
        print(f"\tDone doubling: {result}")
    
        return result

    return wrapper(x)
```

### Decorator: `double` !!!

```python   
def debug(func):

    def wrapper(x):
        print("Doubling ...")
    
        result = func(x)
    
        print(f"\tDone doubling: {result}")
    
        return result

    return wrapper

@debug
def double(x):
    return x*2
```

### Decorator: `double`

```python
def debug(func):

    def wrapper(*args, **kw_args):
        print("Doubling ...")
    
        result = func(*args, **kw_args)
    
        print(f"\tDone doubling: {result}")
    
        return result

    return wrapper

@debug
def double(x):
    return x*2
```
